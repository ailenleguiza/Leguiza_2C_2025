Este archivo contiene la implementación de un sistema basado en Free\+RTOS que mide la distancia utilizando un sensor ultrasónico HC-\/\+SR04 y controla el estado de varios LEDs y una pantalla LCD basada en los valores medidos. Este proyecto está bajo la licencia MIT. Consulta el archivo LICENSE para más detalles.\hypertarget{index_usage}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_usage}
El archivo configura y controla tres LEDs, una pantalla LCD y un sensor ultrasónico HC-\/\+SR04. El sistema crea tres tareas de Free\+RTOS\+:
\begin{DoxyItemize}
\item {\ttfamily medirdistancia}\+: Lee la distancia del sensor ultrasónico.
\item {\ttfamily valorteclas}\+: Lee el estado de los interruptores para cambiar el comportamiento del sistema.
\item {\ttfamily mostrar\+Distancia}\+: Actualiza el estado de los LEDs y la pantalla LCD en función de la distancia medida y el estado de los interruptores.
\end{DoxyItemize}\hypertarget{index_dependencies}{}\doxysection{\texorpdfstring{Dependencies}{Dependencies}}\label{index_dependencies}

\begin{DoxyItemize}

\item Drivers de hardware (LED, interruptores, HC-\/\+SR04, LCD) 
\end{DoxyItemize}
